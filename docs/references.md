# 参考文献

このリポジトリで利用する参考文献の一覧です。

## 使い方
- 公式ドキュメント・書籍・記事をこのファイルに追加する
- 各パターン固有の参考は `catalog/{pattern}/{pattern}.md` にも併記する
- 最低限、タイトルと URL（または書誌情報）を残す

## 参照方針
- もっとも信頼する情報源は `## デザインパターン` セクションとする
- それ以外の参考文献は学習補助として扱う
- 設計判断で迷った場合は `## デザインパターン` を優先する
